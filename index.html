Getting started

Include scripts

Start by including jQuery and the plugin.

				<script src="jquery.js"></script>
				<script src="jquery.inputHistory.js"></script>

Activate the plugin

Now call .inputHistory() on your inputs, like this:

				<script>
					// jQuery ready event
					$(function() {
						$('.target').inputHistory();
					});
				</script>
